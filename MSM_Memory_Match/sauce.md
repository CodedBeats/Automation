Adding Boards and tiles
Board:
    ctrl + prtsc
    ctrl + v in paint
    check proportions are 1920 x 1080
    save as image

tile: 
    prtsc
    select tile area
    ctrl + c
    ctrl + v in paint
    resize
    increment horizontal pixel (+1)
    crop
    save as image
